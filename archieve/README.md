# Archieve

This folder is for confirmed unused or legacy project files that should be retained before deletion.

Rules:

- Move files here only after confirming they are not on the active generation path.
- Prefer keeping a short note beside large moved folders explaining why they were archived.
- Do not archive active runtime assets such as `skills/ppt-master/templates/charts`, `templates/icons`, cookbook files currently referenced by CLI, or scripts used by `api_ppt.py generate`.
